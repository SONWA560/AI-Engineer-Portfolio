## Overview

Think Tank Software Solutions, a South African intelligent tech services provider, was losing competitive ground due to a slow, manual RFP/RFQ review process. Sales teams spent up to five days per document, extracting requirements by hand with inconsistent accuracy. I designed and built a cloud-native AI system to automate that entire pipeline: from document ingestion through to structured bid recommendations.

## The Problem

Every RFP/RFQ cycle exposed the same pain points:

- Sales teams spent **5 days per document** on manual extraction
- Missed requirements led to inaccurate, low-quality proposals
- No integration between document tools and enterprise systems (SharePoint, Ivanti Neurons)
- Growing client volumes were outpacing the team's capacity

## What I Built

The solution is a **cloud-native document intelligence platform** built on Microsoft Azure, with a React frontend for sales teams and a FastAPI backend handling processing logic.

### AI Processing Pipeline

Documents flow through a three-stage pipeline:

1. **Ingestion:** Secure upload to Azure Blob Storage with PII anonymization
2. **Extraction:** Azure Form Recogniser parses structure; Azure Cognitive Search indexes content; OpenAI classifies requirements into technical, financial, and compliance categories
3. **Output:** Structured recommendations and actionable bid insights delivered to the sales team UI

### Integrations

REST APIs connect the platform bidirectionally with SharePoint for document management and Ivanti Neurons for workflow automation. Webhooks push real-time status notifications to sales teams without requiring manual polling.

### Security & Compliance

- OAuth 2.0 + 2FA for access control
- Azure Key Vault for credential management
- Role-based access control (RBAC) with full audit logging
- GDPR/POPIA compliant, with data encrypted at rest and in transit

## Results

**Efficiency**
- Analysis time cut from **5 days to 2 days** per RFP/RFQ
- **60% reduction** in manual effort for document analysis
- **90%+** requirement extraction accuracy

**Business impact**
- **25% improvement** in win rates through higher-quality proposals
- **100% adoption** by the sales team within two weeks of launch

**Frontend:** React, TypeScript  
**Backend:** FastAPI, Python  
**AI/Cloud:** Azure Form Recogniser, Azure Cognitive Search, OpenAI API, Azure Blob Storage, Azure Key Vault
